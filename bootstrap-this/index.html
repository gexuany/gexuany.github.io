About

Introduction

Bootstrap is a modular web design tool that can be used quickly to design web pages with a good layout and functions. Bootstrap has a wide variety of preset styles, which meets all the effects of creating a website.

History

Bootstrap, originally named Twitter Blueprint, was developed by Mark Otto and Jacob Thornton at Twitter. They build is as a framework to encourage consistency across internal tools.The development of Bootstrap has gone through 5 versions. Bootstrap 5 was officially released on May 5, 2021. It has undergone many improvements and adjustments.

Features

Open resource

Bootstrap is an open-code resource that can be used for free so that everyone can participate in adjustment and development.

Mobile First

The developer considers mobile-first so that the user's design is more adapted to the mobile interface.

Components library

Bootstrap has a rich library of elements, allowing users to call and adjust quickly. And these elements are responsive.

Components library

Buttons

Bootstrap's custom buttons have many looks, with many functions. In addition, the user can use and adjust the style of the controller.

Cards

Bootstrap's cards provide a responsive container, which can change different styles and give users many choices.

Carousel

Carousel is a slideshow component, which can conveniently provide users with the banner design on the web page.
